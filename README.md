# snapp-challenge

For Deploy Application with Nginx run command :

````
git clone https://github.com/sdmoradi/snapp-challenge.git

cd snapp-challenge

docker compose up -d
````

For Deploy monitoring stack run command :

````
cd monitoring 

docker compose up -d
````

### Application address:

- http://localhost

### Grafana address:

- http://localhost:3000

### Prometheus address:

- http://localhost:9090
